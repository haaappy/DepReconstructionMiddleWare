    类加载平台用户使用手册

    前言

    本文为类加载平台用户手册,为使用该工具的用户提供帮助。

    环境配置

    • OS: Ubuntu 12.04
    • JDK: Java Version “1.6.0 24”
    • IDE: Eclipse IDE for Java EE Developers

    使用

    运行代码，启动平台的客户端图形界面。

    输入参数

    • 部署目录: 类加载平台的部署目录
    • 文件目录: 需要执行操作的文件路径，可以通过按钮选择文件
    • 部署方式：普通文件夹形式和Jar包形式部署
    • 类加载器：5种类加载器的选择
    • 轮询时间：2秒、5秒和8秒的部署扫描时间的选择

    功能按钮

    • 启动：启动选择的类加载平台
    • 停止：停止选择的类加载平台
    • 选择目录：部署目录选择
    • 选择文件：文件路径选择
    • 部署文件：在平台的部署目录中，复制所选文件，进行部署
    • 卸载文件：在平台的部署目录中，删除所选文件，进行卸载

    控制台信息

    将输出平台的运行信息。

    已加载的类信息

    平台将显示已经加载的类的全限定名称以及它们的加载时间。
